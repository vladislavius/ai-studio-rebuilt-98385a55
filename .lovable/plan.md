

# ИИ-методист — генерация курсов по технологии Хаббарда

## Идея

Добавить в редактор контрольного листа (CourseChecksheet) кнопку **"Сгенерировать с ИИ"**. Админ заполняет форму с параметрами курса (название, аудитория, конечный продукт, модули), нажимает кнопку — edge function вызывает AI-модель с системным промптом методиста — получает структурированный чекшит — пункты автоматически заполняются в редакторе.

## Реализация

### 1. Edge function `generate-checksheet`
- Принимает JSON с параметрами курса (title, audience, final_product, modules)
- Системный промпт = полный текст методиста из сообщения пользователя
- Модель: `google/gemini-2.5-flash` (баланс скорости и качества)
- Возвращает массив checksheet items в формате `{type, title, content}`
- AI отвечает JSON-массивом, edge function парсит и возвращает клиенту

### 2. UI: модалка генерации в CourseChecksheet
- Кнопка "Сгенерировать с ИИ" рядом с "Добавить пункт"
- Модалка с полями:
  - Целевая аудитория (textarea)
  - Конечный продукт / компетенция (textarea)
  - Модули (динамический список: название + темы + уровень сложности)
  - Длительность в неделях
- Кнопка "Сгенерировать" → вызов edge function → заполнение items[]
- Существующие пункты заменяются (с подтверждением если не пустые)

### 3. Расширение типов заданий
Текущие типы: `read`, `write`, `demo`, `drill`, `starrate`
Добавить: `clay_demo` (глиняное демо), `checkout` (чек-аут), `word_clearing` (прояснение слов)
— обновить `TYPE_LABELS` в CourseChecksheet и `TYPE_META` в CourseStudyView
