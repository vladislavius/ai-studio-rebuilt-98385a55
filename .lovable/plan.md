

## Проблема

Сейчас учебный материал (текст статьи) привязан к каждому пункту контрольного листа. Чтобы добавить материал, админ должен:
1. Открыть курс (клик по карточке)
2. Добавить пункт в контрольный лист
3. Нажать маленькую стрелку **▶** справа от заголовка пункта — только тогда раскрывается WYSIWYG-редактор с полем «Учебный материал»

Проблема: эта стрелка практически незаметна, и неочевидно, что внутри каждого пункта скрыт полноценный текстовый редактор для материала.

## План улучшений

### 1. Сделать редактор материала видимым по умолчанию
- Пункты контрольного листа раскрываются автоматически при создании или при клике на пункт
- Добавить явную кнопку «Добавить материал» если контент пуст, вместо скрытого chevron

### 2. Визуально выделить зону материала
- Добавить заметный индикатор на каждом пункте: «Материал добавлен ✓» или «⚠ Нет материала»
- Кнопка «Редактировать материал» с иконкой вместо невидимой стрелки

### 3. Улучшить UX редактора пункта (ChecksheetItemEditor)
- Заменить маленький chevron на явную кнопку «Развернуть / Редактировать содержимое»
- При пустом `content` показывать предупреждение: «Добавьте учебный материал для этого шага»
- Автоматически раскрывать новые пункты
